Bu repo, temel HTTP esaslarını kapsayan pratik notlar ve terminal günlükleri içerir.

Temel HTTP metotlarını (GET, POST, PUT, PATCH, DELETE) ve standart yanıt durum kodlarını (2xx, 4xx, 5xx) belgelendirir.
Ayrıca, JSONPlaceholder API'si kullanılarak yapılan gerçek dünya cURL istek örneklerini ve yanıt başlığı (header) analizlerini içerir.
